# Roadmap

## Phase 1: 構造定義

- [x] プロジェクトの目的と対象を定義する
- [x] 最適化対象を定義する
- [x] 英語能力を9領域へ分解する

## Phase 2: 診断設計

- [x] 各能力を測る問題形式を定義する
- [x] 制限時間あり・なしの測定方法を定義する
- [x] 正答率、速度、確信度、再現率の尺度を作る
- [x] 初回診断の所要時間と実施順を決める
- [x] 診断結果の出力形式を作る
- [ ] 著作権上問題のない実問題を作成する
- [ ] 問題難易度を等化する
- [ ] 暫定レベル境界を実データで校正する
- [ ] 再検査信頼性と予測妥当性を確認する

設計仕様は完了していますが、実問題と評価尺度は未検証です。現段階のLevel 0から4は合否判定に使いません。
