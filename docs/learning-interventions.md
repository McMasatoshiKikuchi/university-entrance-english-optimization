# 能力別学習介入

診断されたボトルネックに対し、学習行動と成功条件を対応させます。「教材を使った」ではなく、能力測定値が改善したかで終了を判断します。

| LEX | 単語帳では分かるが文中で誤る | 文脈語義、品詞、コロケーション確認 | 単独と文中の差 | 類題で転用できる |

## 共通手順

1. 診断結果から一つの下位能力を選ぶ。
2. 介入前の基準値を記録する。
3. 1週間に3から5回、短い反復を行う。
4. 学習に使っていない類題で再テストする。
5. 成功条件を2回連続で満たしたら維持へ移す。
6. 2週間改善しなければ、原因・難易度・方法を再診断する。

## 学習行動の品質条件

- 正解を読むだけで終わらず、想起または生成を含む。
- 同じ問題の暗記と、能力の転用を区別する。
- 学習直後だけでなく、時間を空けた再テストを行う。
- 時間を測る場合も、理解度を犠牲にした速度を成功としない。
- 解説が理解できない教材は、難易度または前提能力を見直す。
- 音声・発話の録音を保存する場合は本人同意と保存期間を決める。
