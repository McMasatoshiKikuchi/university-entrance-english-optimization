# 一般利用者向け利用案内

## 1. このサイトについて

このサイトは、日本の大学受験英語について、現在の能力、志望校の出題形式、残り期間、学習時間を整理し、学習の優先順位を決めるための公開資料です。

閲覧にGitHubアカウントは必要ありません。資料を読むだけであれば、GitHubへのログインやIssueの作成は不要です。

## 2. 推奨する読み進め方

1. [大学受験英語 能力マップ](ability-map.md)で、英語力を10領域に分けて確認します。
2. [初回診断テスト仕様](diagnostic-test-spec.md)で、現在のボトルネックを調べる考え方を確認します。
3. [入試方式と要求能力の対応](exam-type-requirements.md)で、志望方式に必要な能力を確認します。
4. [学習優先度モデル](priority-model.md)で、何から学ぶかを決めます。
5. [残り期間別ロードマップ](time-horizon-roadmaps.md)と[週次再計画ルール](weekly-replanning.md)で、計画を更新します。

## 3. テンプレートの使い方

テンプレートは、そのまま記入するためのひな型です。GitHub上で編集する必要はありません。内容を手元のノート、文書、表計算ソフトなどへ転記して利用できます。

- [診断結果テンプレート](../templates/diagnostic-result.md)
- [大学別分析テンプレート](../templates/university-exam-profile.md)
- [週間最適化テンプレート](../templates/weekly-optimization.md)
- [過去問レビューテンプレート](../templates/past-paper-review.md)
- [模試レビューテンプレート](../templates/mock-exam-review.md)

## 4. データ表について

CSVは、能力コード、診断結果、学習記録などを表形式で保存するためのデータです。閲覧だけで学習を開始できるため、必ずしもCSVを使用する必要はありません。

CSVを利用する場合は、表計算ソフトで開き、見本の行を複製して記録します。列名は変更せず、個人を特定できる情報は入力しないでください。

## 5. GitHubアカウントが必要な機能

次の操作にはGitHubアカウントとログインが必要です。

- Issueを作成して学習実験を登録する
- 資料の修正を提案する
- リポジトリへファイルを追加・変更する
- 更新通知を受け取る

これらは主に運営者・共同編集者向けです。一般利用者がPagesを閲覧するだけなら必要ありません。

## 6. 個人情報と著作権

入力・公開しないもの:

- 氏名、生年月日、学校名、受験番号、連絡先
- 顔写真、音声、動画、健康・家庭に関する情報
- パスワード、認証コード、アクセストークン
- 市販教材、模試、過去問の問題本文・画像・PDF
- 本人の同意がない第三者の情報

記録が必要な場合は、実名ではなく本人だけが対応関係を管理できる仮のIDを使用してください。

## 7. この資料の限界

優先度係数、評価尺度、必要学習時間は、現段階では検証途中の暫定値を含みます。合否判定や進路決定を自動化するものではありません。大学の募集要項、配点、試験方式は、必ず大学公式の最新情報で確認してください。
