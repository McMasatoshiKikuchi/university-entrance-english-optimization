# 教材選定基準

## 1. 原則

教材は評判ではなく、診断された能力、志望校形式、残り期間、完了可能性から選びます。教材数を増やすことは最適化ではありません。

## 2. 必須条件

次のいずれかを満たさない教材は、点数評価の前に候補から外します。

- 著作権・利用条件に問題がない。
- 対象能力が明確である。
- 解答・解説または適切なフィードバック手段がある。
- 残り期間内に必要範囲を完了できる。
- 現在レベルから取り組める。
- 入試形式または前提能力へ接続できる。

## 3. 評価式

各項目を0から5で評価します。

| 項目 | 重み | 確認内容 |
| --- | ---: | --- |
| bottleneck_fit | 25 | 診断された弱点へ直接対応する |
| level_fit | 15 | 難しすぎず簡単すぎない |
| exam_fit | 15 | 志望校の形式・配点へ接続する |
| feedback | 15 | 誤りの原因と修正方法を確認できる |
| completion | 15 | 残り期間と週間時間で完了できる |
| transfer | 10 | 他の問題・大学にも転用できる |
| usability | 5 | 継続しやすく記録可能である |

```text
material_score
= Σ(項目得点 / 5 × 項目重み)
```

100点満点ですが、得点差が小さい場合は新規購入せず、現在の教材を完了できる方を優先します。

## 4. 教材の役割

同一能力について、原則として次の役割を重複させません。

- input: 知識・方法を理解する
- guided_practice: 解説を参照しながら適用する
- independent_practice: 自力で適用する
- timed_practice: 制限時間内で実行する
- assessment: 未使用問題で測定する
- review: 誤答と再テストを管理する

assessment用の問題を事前学習へ使い切らないよう分離します。

## 5. 採用・停止ルール

### 採用

- 対応する能力コードを記録する。
- 完了範囲と完了予定日を決める。
- 週間使用時間を決める。
- 使用前の基準値を記録する。
- 成功条件と再テストを決める。

### 一時停止

- 2週連続で計画の50%未満しか進まない。
- 解説理解に必要な前提能力が不足している。
- 同じ役割の教材が重複している。
- 入試方式の変更で重要度が下がった。

### 終了

- 予定範囲を完了し、未使用類題で成功条件を2回満たした。
- より高いレベルの課題へ移る。
- 教材の目的が志望校と一致しないことが判明した。

教材を途中で変えた場合は、理由を記録します。感覚的な変更を繰り返しません。

## 6. 過去問の位置付け

過去問は教材であると同時に測定手段です。

- 最初の1年分: 形式把握
- 中間の年度: 学習効果の測定
- 直近年度: 本番シミュレーション用に温存する候補

問題本文や画像はリポジトリへ転載せず、年度、形式、得点、時間、失点原因、次の行動だけを記録します。
