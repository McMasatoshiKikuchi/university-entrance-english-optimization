# 入試方式と要求能力の対応 v0.1

## 1. この文書の位置付け

大学受験英語を、学校区分ではなく実際の問題形式と評価方法から分類します。

同じ「国公立」「私立」でも、学部・方式・年度により問題形式と配点が異なります。この文書の能力重みは学習計画を作るための暫定仮説であり、大学が公表する公式配点ではありません。個別大学の募集要項、試験時間、過去問を確認して上書きします。

能力重み:

- 0: 直接は評価されない
- 1: 影響が小さい、または方式により課される
- 2: 補助的
- 3: 中程度
- 4: 重要
- 5: 中核

METAは試験で直接採点される能力ではないため、マトリクスから除外し、学習運用側で常に管理します。

## 2. 共通テスト英語

令和9年度共通テスト英語は、リーディング80分・100点、リスニングは解答時間30分・100点です。原則として両方を受験します。大学入試センターは、目的・場面・状況に応じて、概要、要点、詳細、話し手・書き手の意図を理解する力、情報や考えを整理・判断する力、語彙・表現・文法等を活用する力を重視しています。

### CT-R: 共通テスト リーディング

重点:

- RDN: 複数形式の文章・資料を読み、目的に応じて情報を処理する
- QST: 設問要求、言い換え、根拠、複数資料の対応
- EXE: 80分内の大問配分、保留、後半の集中維持
- LEX: 文脈内で語彙を高速認識する

文法知識は独立問題よりも、読解と情報処理を支える能力として扱います。WRT・SPKは直接記述・発話させる形式ではないため、直接重みを0とします。

### CT-L: 共通テスト リスニング

重点:

- LST: 音声認識、情報保持、意図理解、資料統合
- QST: 先読み、選択肢比較、条件処理
- EXE: 1回読み・2回読みへの切替、メモ、集中維持
- LEX: 音声から語彙を即時認識する

多様な話者による現代の標準的な英語が使われ、1回読みと2回読みの両方が含まれます。

### CT-U: 共通テスト利用・併用

要求能力はCT-R・CT-Lと同じですが、大学ごとに次が異なります。

- リーディングとリスニングの換算比率
- 英語の総合配点
- 必要科目数
- 一定点を満たす資格判定か、合計点への加算か
- 個別試験との併用方法

共通テスト本体の100対100と、大学が合否判定に使う換算比率を混同しません。

## 3. 国公立大学の個別試験

### NAT-W: 記述・論述中心型

代表的な形式:

- 英文和訳
- 内容説明
- 要約
- 和文英訳
- 自由英作文
- 長文の根拠を用いた記述

重点:

- SYN: 複雑な文構造を正確に解析する
- RDN: 論旨と段落構造を把握する
- WRT: 必要要素を過不足なく表現する
- GRM: 読解・記述の両方で正確に運用する
- QST: 設問の条件と採点要素を満たす

文部科学省の令和9年度実施要項は、個別テストで客観式と記述式を組み合わせ、思考・判断の過程や結果を表現する能力を評価する工夫を推奨しています。また、共通テストと同じ科目を個別試験でも課す場合は、共通テストとは異なる能力判定へ力点を置くことが望ましいとしています。

### NAT-M: 総合問題・資料統合型

英語単独ではなく、複数資料、専門分野、小論文的課題などと統合される形式です。

重点:

- RDN-08: 図表・複数資料の統合
- RDN-05 / 06: 要旨・推論
- WRT-04 / 07: 構成・要約
- QST-06: 記述条件
- EXE: 長い処理工程の時間管理

## 4. 私立大学一般選抜

私立は大学名ではなく、学部・方式ごとに分類します。

### PVT-R: 長文・内容一致中心型

重点:

- LEX: 高い語彙水準と認識速度
- RDN: 長文量、論旨、情報検索
- QST: 内容一致、空所、言い換え、推論
- EXE: 大量処理と捨て問判断

### PVT-G: 語彙・文法・語法中心型

重点:

- LEX: 語義、熟語、派生語、コロケーション
- GRM: 文法形式、語法、正誤、整序
- QST: 短時間での形式判定
- EXE: 高速処理と見直し

この型でも長文が併存する場合が多いため、RDNを完全には外しません。

### PVT-I: 長文・文法・記述の統合型

長文読解に加えて、英文完成、整序、和訳、要約、英作文などを組み合わせる形式です。

重点:

- LEX・GRM・SYN・RDNを横断する運用
- QSTによる形式切替
- WRTによる短い記述
- EXEによる大問間の配分

### PVT-E: 英語外部試験利用型

活用方法を次のように分けます。

- 出願資格
- 英語試験の免除
- 得点換算
- 加点
- 当日試験との高得点採用
- 共通テスト・個別試験との併用

必要能力は利用する資格・検定の仕様に従います。4技能試験の場合はLEX・GRM・RDN・LST・WRT・SPKを対象にします。有効期限、対象級・スコア、換算方法、証明書提出方法を大学公式情報で確認します。

## 5. 総合型・学校推薦型選抜

令和9年度の文部科学省実施要項では、総合型選抜と学校推薦型選抜について、多面的・総合的な評価を行い、原則として面接による評価を実施する枠組みが示されています。また、調査書等だけでなく、大学教育に必要な知識・技能、思考力・判断力・表現力等を評価する方法を少なくとも一つ活用するよう示されています。

英語に関係する評価方法:

- 英語資格・検定
- 英語による面接・口頭試問
- 英文資料を含む小論文・総合問題
- 英語プレゼンテーション
- 志望理由書等に基づく質疑
- 共通テスト英語
- 大学独自の英語試験

### REC-Q: 資格・検定利用型

4技能試験では、LST・RDN・WRT・SPKを直接評価し、LEX・GRMが基盤になります。

### REC-O: 英語面接・口頭試問型

重点:

- SPK: 応答、説明、論理構成、聞き返し・修復
- LST: 質問理解、意図把握
- LEX / GRM: 意味を通すための運用
- EXE: 緊張下での応答管理

流暢さだけでなく、質問に答えているか、根拠を示せるか、分からない場合に修復できるかを測ります。

## 7. 個別大学へのマッピング手順

1. 大学公式の入試変更予告、入学者選抜要項、募集要項を取得する。
2. 大学名ではなく、学部・学科・選抜区分・日程でレコードを分ける。
3. 英語の配点、換算、試験時間、問題形式を記録する。
4. 過去問から大問別形式と分量を記録する。
5. 各形式を能力コードへ対応させる。
6. 能力重み0から5と根拠を登録する。
7. 公式情報と過去問分析を区別する。
8. 年度が変わるたびに再確認する。

令和9年度について、各大学は選抜区分ごとの基本事項を令和8年5月27日から7月31日までに発表し、募集要項は令和8年12月15日までに発表する枠組みです。早期情報だけで確定扱いせず、募集要項公開後に再確認します。
