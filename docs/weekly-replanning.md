# 週次再計画ルール

## 1. 週の開始時

1. 志望校・方式と残り日数を更新する。
2. 診断・模試・過去問の新しい結果を追加する。
3. 能力別priority_scoreを再計算する。
4. 重点能力を2から3個選ぶ。
5. 学習時間を分単位で配分する。
6. 各課題に成功条件と再テスト日を付ける。
7. 今週行わないことを決める。

## 2. タスクの書き方

悪い例:

- 英単語を頑張る
- 長文を進める
- リスニングを復習する

良い例:

- LEX-02: 文脈語義20問を10分で実施し、翌日類題で85%以上
- RDN-02: 2長文の各段落要旨を一文で記録し、本文の役割と照合
- QST-04: 誤答選択肢10件を過剰・部分一致・本文外へ分類
- LST-03: 初見音声とスクリプト確認後の語認識率の差を記録

## 3. 週末レビュー

| 確認 | 判断 |
| --- | --- |
| 完了率 | 予定時間ではなく成功条件まで完了した割合 |
| 能力変化 | 再テストで正答率・時間・再現が改善したか |
| 転用 | 未使用類題でも改善したか |
| 得点寄与 | 模試・過去問で回収点が増えたか |
| 計画精度 | 予定時間と実績時間の差 |
| 継続性 | 睡眠・学校生活を破壊していないか |

## 4. 自動的な変更ルール

- 完了率60%未満: 翌週の計画量を20から30%減らす。
- 予定時間を毎回超える: hours_to_next_levelと教材難易度を見直す。
- 成功条件を2回連続達成: 維持課題へ移し、時間を減らす。
- 同じ失点が2回続く: 誤り分類と前提能力を再診断する。
- 同じ介入で2週間改善なし: 方法・難易度・フィードバックを変更する。
- priority上位が4個以上: 上位2から3個へ絞り、残りは保留する。
- 本番30日以下: 本番形式比率を55%程度まで上げる。
- 公式の入試変更: exam_importanceとtarget_levelを即時更新する。
- 体調不良・睡眠不足: 最低ラインへ縮小し、取り返し計画を積み上げない。

## 5. 維持課題

目標レベルへ到達した能力を完全に停止すると低下する場合があります。

- LEX: 少量の間隔反復
- LST: 週1から2回の初見音声
- WRT / SPK: 週1回の短い生成
- EXE: 定期的な本番形式
- 過去に再発した弱点: 週末の短い再テスト

維持課題は全体の10から20%を上限の目安とし、主要ボトルネックを圧迫しないようにします。

## 6. 判断記録

数式と異なる配分を選んだ場合は、次を記録します。

- 変更した能力・時間
- 数式上の順位
- 上書き理由
- 期待する効果
- 判断を再確認する日

これにより、結果が良かった場合も悪かった場合も判断方法を改善できます。
