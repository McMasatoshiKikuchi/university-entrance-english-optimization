# 初期診断ルール

この文書は、観察された失点を能力領域へ仮分類するための初期ルールです。確定診断ではなく、追加測定の入口として使います。

## 代表的な判定

| 観察 | 第一候補 | 追加確認 |
| --- | --- | --- |
| 単語で頻繁に停止する | LEX-07 | 単語単独と文中の回答時間を比較 |
| 単語帳では分かるが長文で分からない | LEX-02 / GRM-11 | 文脈語義と品詞を確認 |
| 単語は分かるが一文を誤読する | GRM / SYN | 主語・動詞・修飾関係を説明させる |
| 短文は読めるが長文の主張が分からない | RDN-02 / RDN-05 | 段落要旨を一文でまとめる |
| 内容は説明できるが選択問題を落とす | QST | 根拠箇所と誤答理由を示させる |
| 時間無制限なら高得点 | RDN-09 / EXE | 大問別の所要時間を測る |
| 後半だけ急激に正答率が下がる | RDN-10 / EXE-06 | 疲労と時間圧迫を分ける |
| 読める英文を聞き取れない | LST-01から03 | スクリプト確認前後の理解を比較 |
| 聞き取れるが選択肢を選べない | LST-04から07 / QST | 情報保持と資料統合を確認 |
| 内容は考えられるが英文にできない | WRT-01から03 | 日本語簡略化と基本文生成を分ける |
| 英文は書けるが点が伸びない | WRT-04から06 | 設問条件、論理、語数を確認 |
| 同じ間違いを繰り返す | META-01から03 | 復習記録と再テストを確認 |

## 診断の原則

1. 一つの失点を一つの原因に即断しない。
2. 知識の有無と、時間内に使えるかを分ける。
3. 単独問題と長文内での運用を分ける。
4. 初見、解き直し、再テストを区別する。
5. 正答でも偶然や消去法だけの場合は未定着として記録する。
6. 本文理解と設問処理を分ける。
7. 得点だけでなく、所要時間と確信度を残す。

## 記録する最小項目

- 実施日
- 問題形式
- 能力コード
- 正誤
- 所要時間
- 確信度
- 失点原因
- 次の学習行動
- 再テスト日
- 再テスト結果

## 次段階で定義するもの

- 能力別の測定問題仕様
- 標準的な実施時間
- 正答率・速度・再現率の評価尺度
- 入試方式ごとの必要能力の重み
- 診断結果から学習課題を選ぶ決定ツリー
